O que é Python e por que aprender?

Python é uma das linguagens de programação mais populares do mundo. Ela foi criada em 1991 pelo programador holandês Guido van Rossum e recebeu esse nome em homenagem ao grupo de comédia britânico Monty Python.

Por que Python é tão amada?

  • ✅ Tem uma sintaxe limpa — parece quase falar em inglês
  • ✅ É fácil para iniciantes aprendem lógica de programação
  • ✅ É poderosa o suficiente para criar sites, aplicativos, análise de dados e inteligência artificial
  • ✅ Tem uma comunidade enorme — sempre tem alguém para ajudar
  • ✅ É gratuita e funciona em Windows, Mac e Linux

Para que serve Python?

ÁreaExemplos do mundo real
🌐 Sites e aplicações webInstagram, Pinterest e Spotify usam Python
🤖 Inteligência ArtificialChatbots, reconhecimento de imagens
📊 Análise de dadosGráficos, relatórios, Excel automatizado
🎮 JogosPyGame para criar jogos 2D
🔬 CiênciaCálculos científicos e simulações
⚙️ AutomaçãoTarefas automáticas no computador

Como instalar o Python?

Acesse python.org/downloads e baixe a versão mais recente. Durante a instalação, marque a opção "Add Python to PATH" — isso é muito importante!

💡 Dica de iniciante: Você não precisa instalar nada agora para acompanhar esta apostila! Pode usar o python.org/shell diretamente no navegador para praticar.

Depois de instalado, abra o terminal (ou o Prompt de Comando no Windows) e digite:

python --versionterminal

Se aparecer algo como Python 3.12.0, está funcionando! 🎉

Primeiro programa: Olá, Mundo!

Todo programador começa pelo mesmo lugar: o famoso Olá, Mundo! É uma tradição mostrar que o programa está funcionando exibindo essa mensagem na tela.

Em Python, isso é incrivelmente simples:

print("Olá, Mundo!")
Python

Quando você rodar esse código, verá no terminal:

Olá, Mundo!resultado

O que é print()?

O print() é uma função — um comando que diz ao Python para exibir algo na tela. Tudo que você colocar entre os parênteses será mostrado.

print("Meu nome é Python")
print("Eu sou uma linguagem incrível!")
print(2026)
print(3.14)
Python
Meu nome é Python
Eu sou uma linguagem incrível!
2026
3.14resultado
⚠️ Atenção: Em Python, textos precisam estar entre aspas ("assim" ou 'assim'). Números ficam sem aspas. Se errar isso, o programa vai reclamar!

Comentários — notas dentro do código

Comentários são anotações que você escreve para você mesmo (ou para outros que lerem seu código). O Python ignora completamente tudo que vem depois do símbolo #.

# Este é um comentário — o Python ignora esta linha
print("Esta linha sim é executada")  # comentário no final também funciona
Python

Regras para nomear variáveis

  • ✅ Pode usar letras, números e underline: minha_variavel, nome2

  • ❌ Não pode começar com número: 2nome é inválido

Juntando textos — concatenação e f-strings

Entrada de Dados pelo Usuário

Até agora, os dados do programa eram definidos por você no código. Mas programas de verdade pedem informações ao usuário. Para isso, usamos a função input().

nome = input("Qual é o seu nome? ")
print(f"Olá, {nome}! Seja bem-vindo(a) ao Python!")
Python

Olá, Maria! Seja bem-vindo(a) ao Python!resultado

⚠️ Input sempre retorna texto!

Um ponto muito importante: tudo que o usuário digita vem como texto (str), mesmo que seja um número. Se quiser fazer cálculos, precisa converter:

# ERRADO — tentando somar sem converter
numero = input("Digite um número: ")
print(numero + 10)  # ERRO! não dá para somar texto com número

# CORRETO — convertendo para inteiro com int()
numero = int(input("Digite um número: "))
print(numero + 10)  # funciona!
Python
FunçãoConverte paraExemplo

float()Número decimalfloat("3.14") → 3.14
str()Textostr(42) → "42"

Condicionais — if / elif / else

A vida está cheia de condições: se estiver chovendo, pego o guarda-chuva; senão, saio sem ele. Em programação é igual — condicionais permitem que seu programa tome decisões.

Estrutura básica

if condicao:
    # código executado se a condição for verdadeira
elif outra_condicao:
    # executado se a primeira for falsa, mas esta for verdadeira
else:
    # executado se nenhuma condição anterior for verdadeira
Python
⚠️ Indentação é OBRIGATÓRIA! Em Python, o código dentro de um if precisa estar recuado (com espaços ou tab). Use 4 espaços — é o padrão. Se não indentar, o Python vai reclamar!

Exemplo 1 — verificar maioridade

idade = int(input("Quantos anos você tem? "))

if idade >= 18:
    print("Você é maior de idade! ✅")
else:
    print("Você é menor de idade. ❌")
Python

Exemplo 2 — classificação escolar

nota = float(input("Digite sua nota (0 a 10): "))

if nota >= 9:
    print("🏆 Excelente!")
elif nota >= 7:
    print("✅ Aprovado(a)!")
elif nota >= 5:
    print("⚠️ Recuperação...")
else:
    print("❌ Reprovado(a).")
Python

Condições combinadas

temperatura = 35
tem_sol = True

if temperatura > 30 and tem_sol:
    print("Dia perfeito para a praia! 🏖️")
elif temperatura > 30 and not tem_sol:
    print("Calor, mas sem sol. Pode sair! ☁️")
else:
    print("Melhor ficar em casa. 🏠")
Python

Repetição — for e while

Imagine pedir para um aluno copiar uma frase 100 vezes. Você não escreveria o print() 100 vezes no código — usaria um loop (laço de repetição).

O loop for — repetição com contagem

Use o for quando você sabe quantas vezes quer repetir algo:

for i in range(5):
    print(f"Repetição número {i}")
Python

ℹ️ range(): O range(5) gera os números 0, 1, 2, 3, 4 — começa no 0 e para antes do 5. Use range(1, 6) para ir de 1 a 5. Use range(0, 10, 2) para ir de 2 em 2.

Percorrendo uma lista com for

frutas = ["maçã", "banana", "laranja", "uva"]

for fruta in frutas:

O loop while — repetição com condição

Use o while quando quer repetir enquanto uma condição for verdadeira:

tentativas = 0
senha_correta = "python123"

while tentativas < 3:
    senha = input("Digite a senha: ")
    if senha == senha_correta:
        print("✅ Acesso liberado!")
        break  # sai do loop imediatamente
    else:
        tentativas += 1
        print(f"❌ Senha errada. Tentativa {tentativas}/3")

if tentativas == 3:
    print("🔒 Acesso bloqueado.")
Python
⚠️ Cuidado com loop infinito! Se a condição do while nunca ficar falsa, o programa trava. Sempre certifique-se de que a condição vai mudar em algum momento, ou use break para sair.

Funções

Imagine que você precisa calcular o dobro de um número várias vezes no seu programa. Em vez de escrever o mesmo código repetido, você cria uma função — um bloco de código reutilizável com um nome.

Criando sua primeira função

def saudacao():
    print("Olá! Seja bem-vindo(a)!")
    print("Estou feliz que você está aprendendo Python 🐍")

# Chamando (usando) a função
saudacao()
saudacao()  # pode chamar quantas vezes quiser!
Python

Funções com parâmetros

Parâmetros são informações que você passa para a função:

def saudacao_personalizada(nome):
    print(f"Olá, {nome}! Bem-vindo(a)! 👋")

saudacao_personalizada("Maria")
saudacao_personalizada("João")
saudacao_personalizada("Ana")
Python
Olá, Maria! Bem-vindo(a)! 👋
Olá, João! Bem-vindo(a)! 👋
Olá, Ana! Bem-vindo(a)! 👋resultado

Funções que devolvem um resultado — return

def calcular_media(nota1, nota2, nota3):
    media = (nota1 + nota2 + nota3) / 3
    return media

# Usando a função
resultado = calcular_media(8, 7.5, 9)
print(f"Sua média é: {resultado:.1f}")  # .1f = 1 casa decimal
Python
Sua média é: 8.2resultado
💡 Regra de ouro: Se um bloco de código se repete mais de 2 vezes, crie uma função. Código bem organizado em funções é muito mais fácil de corrigir e entender.

Listas

Uma lista é uma coleção de itens organizados em sequência. Imagine uma lista de compras, uma turma de alunos, os filmes que você quer assistir — todos são exemplos de listas.

# Criando uma lista
alunos = ["Maria", "João", "Ana", "Pedro"]

print(alunos)            # lista inteira
print(alunos[0])         # primeiro item (índice 0)
print(alunos[-1])        # último item
print(len(alunos))       # quantidade de itens
Python
['Maria', 'João', 'Ana', 'Pedro']
Maria

ℹ️ Índices começam em 0! O primeiro item está na posição 0, o segundo na posição 1, e assim por diante. Índices negativos contam do fim: -1 é o último, -2 é o penúltimo.

Modificando listas

frutas = ["maçã", "banana"]

frutas.append("uva")       # adiciona no final
print(frutas)  # ['maçã', 'banana', 'uva']

frutas.remove("banana")    # remove pelo valor
print(frutas)  # ['maçã', 'uva']

frutas[0] = "manga"         # altera o primeiro item
print(frutas)  # ['manga', 'uva']

frutas.sort()               # ordena em ordem alfabética

Dicionários

Enquanto listas guardam itens em sequência (posição 0, 1, 2...), dicionários guardam informações com chave e valor — como um dicionário de palavras onde cada palavra tem sua definição.

aluno = {
    "nome": "Maria",
    "idade": 17,
    "nota": 8.5,
    "aprovado": True
}

print(aluno["nome"])     # Maria
print(aluno["nota"])     # 8.5
Python

Adicionando e modificando dados

aluno["email"] = "maria@escola.com"   # adiciona nova chave
aluno["nota"] = 9.0                    # atualiza valor existente

print(aluno)
Python

Verificando se uma chave existe

if "email" in aluno:

Python

🥉 Exercício 1 — Calculadora de IMC

Crie um programa que:

  1. Peça o nome, o peso (kg) e a altura (m) do usuário
  2. Calcule o IMC: peso / (altura ** 2)
  3. Mostre a classificação: abaixo do peso (<18.5), normal (18.5–24.9), sobrepeso (25–29.9), obesidade (≥30)
nome = input("Seu nome: ")
peso = float(input("Peso (kg): "))
altura = float(input("Altura (m): "))

imc = peso / (altura ** 2)

if imc < 18.5:
    classificacao = "Abaixo do peso"
elif imc < 25:
    classificacao = "Peso normal ✅"
elif imc < 30:
    classificacao = "Sobrepeso"
else:
    classificacao = "Obesidade"

print(f"\n{nome}, seu IMC é {imc:.1f}: {classificacao}")
Solução

🥇 Exercício 3 — Jogo de Adivinhação

Crie um jogo onde:

  1. O programa "pensa" em um número secreto (pode começar com um fixo, como 42)
  2. O usuário tem 5 tentativas para adivinhar
  3. A cada tentativa, o programa diz se o número é maior ou menor
  4. Se acertar, parabeniza. Se errar todas, revela a resposta.
numero_secreto = 42
tentativas_maximas = 5
acertou = False

print("🎮 Jogo de Adivinhação!")
print(f"Você tem {tentativas_maximas} tentativas para adivinhar o número (entre 1 e 100).")

for tentativa in range(1, tentativas_maximas + 1):
    palpite = int(input(f"\nTentativa {tentativa}/{tentativas_maximas}: "))

    if palpite == numero_secreto:
        print(f"🎉 Parabéns! Você acertou em {tentativa} tentativa(s)!")
        acertou = True
        break
    elif palpite < numero_secreto:
        print("📈 O número é MAIOR!")
    else:
        print("📉 O número é MENOR!")

if not acertou:
    print(f"\n😅 Acabaram as tentativas! O número era {numero_secreto}.")
Solução
